# LineFlow
Streamlining long, basic drawing workflows

состав: Ивашина Алиса
тема: создание приложения для оптимизации работы в графических редакторах на IOS
концепция: приложение обрабатывает файл с "контуром" и на их основе создает новые с подходящей "заливкой" под входной файл; имитация собственного лайна по входящему шаблону; генерация нового файла с новым цветом относительно того, что был на входном файле
